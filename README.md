This project analyzes Diwali sales data to understand customer purchasing behavior during the festive season. The goal of the analysis is to identify key trends that can help businesses make better data-driven decisions.

The analysis focuses on factors such as gender, age group, state, occupation, and product category. Using these variables, the project highlights high-value customer segments, top-performing states, and popular product categories during Diwali.

A Power BI dashboard is used to visually present insights in a clear and interactive way, making it easier to understand sales patterns and business performance. Python (Pandas) was used for data cleaning and exploratory analysis before visualization.
